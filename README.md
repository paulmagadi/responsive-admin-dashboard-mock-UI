## 📱 Responsive Design
This website adjusts seamlessly to:

✅ Mobile Phones

✅ Tablets

✅ Desktop Screens

It uses flexbox, grid, and media queries for responsiveness.

## 🔧 Tech Stack
- HTML5 – Markup
- CSS3 – Styles & layout
- Vanilla JavaScript – DOM manipulation & logic
- Font Awesome – Icons

